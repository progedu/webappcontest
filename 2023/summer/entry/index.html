コンテストについて

N予備校 動くWebページコンテストは、
N予備校が主催する、
Webページの成果物を競うコンテストです。

応募規定

応募条件、テーマ、作品提出のルールなど

応募条件

作品のテーマは自由ですが、以下のガイドラインに従ってください。

使用可能言語

HTML、CSS、JavaScript

使用可能ライブラリ

BootStrap
BootStrap以外の外部ライブラリは利用しないでください。

動作環境

Google Chrome のブラウザで正常に動作すること。

注意事項

応募作品のコード及びサイトのURLは結果ページにリンクが貼られ、授業にて紹介します。
画像や音声などのメディアを使用する場合は、著作権上問題のないメディアを利用すること。

提出物

応募作品のコードを管理しているGitHubリポジトリのURL。
公開設定はPublicにしてください。

動作確認ができるWebページのURL。
応募作品はGitHubページで公開してください。

応募資格

N・S高 部門

応募時点でN高等学校もしくはS高等学校に在籍していること。

N中等部 部門

応募時点でN中等部に在籍していること。

N予備校 部門

応募時〜結果発表時にN予備校の会員であり、かつ上記の3校に在籍していないこと。

参加費

無料
ただし、N予備校部門の場合、N予備校の会費が必要です。

注意事項

コンテスト期間中に退学・退会した場合は審査対象外となったり賞状や賞品が送付できなくなる可能性があります。

がんばろう

審査

作品の審査基準、審査員について

審査のポイント

以下の3つのポイントで審査します。

コーディング

コードの簡潔さ、可読性、コメントの有無などを評価します。

  • HTML / CSS / JS が役割毎にファイル分割されているか
  • ソースコードに無駄がないか、見やすく書かれているか
  • 入門コース１章で習った要素や技術をうまく使えているか
  • 自分なりに調べて実現したい機能を実装できているか

完成度

作り込みの深さ、バグの少なさ、デザインなどを評価します。

  • 意図した通りに動くか、動作不良やエラーなどはないか
  • 使いやすく、わかりやすい見た目やデザインになっているか
  • 作品のテーマに対して必要な機能が十分に実装されているか

アイデア

創意工夫や発想、テーマに沿っているか、着眼点の良さなどを評価します。

  • 作品のテーマや発想が新しいか、時世やトレンドを取り入れているか
  • コーディングに工夫が見られるか、難しい問題を解決できているか
  • 見せ方に工夫があるか、便利で楽しめるサービスになっているか
